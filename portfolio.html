Capstone Project

Housing Affordability Analysis - Peoria, Arizona

This Power BI dashboard analyzes housing affordability in Peoria, Arizona, with a focus on how home values changed compared to household income over time. The dashboard was created to show whether income growth kept pace with rising home prices.

Business Question

Are homes becoming less affordable in Peoria, Arizona because median home values are increasing faster than median household income?

Key Insights

  • Median home values increased faster than household income over the analysis period.
  • The affordability ratio shows that homes became more expensive compared to local income.
  • The dashboard highlights the growing gap between income and home prices.

Skills & Techniques Used

  • Power BI
  • Power Query
  • Data Modeling
  • DAX Measures
  • Linear Regression
  • KPI Cards
  • Data Storytelling

Business Value

This dashboard helps users understand how affordability changes over time and supports a clearer discussion about housing pressure, income growth, and long-term financial planning.

Power BI Project

Crime in Los Angeles Dashboard

This Power BI dashboard explores Los Angeles crime data from 2020 to 2024. It allows users to filter by year, month, time of day, location, and other categories to better understand crime patterns.

Business Question

When and where are crimes happening most often, and what patterns can be found by time of day, crime type, and location?

Key Insights

  • Time-of-day filters help identify when crime activity is highest.
  • Crime hotspots help users understand which areas may need more attention.
  • Weapon and crime type breakdowns provide more context behind the incidents.

Skills & Techniques Used

  • Power BI
  • Data Cleaning
  • Interactive Filters
  • Map Visuals
  • Time Analysis
  • KPI Reporting
  • Tooltips

Design Improvements

A stronger version of this dashboard should define time categories such as morning, afternoon, evening, and night. It should also include total crimes, percentages by time of day, and clear tooltip explanations.

Tableau Project

British Airways Review Dashboard

This Tableau dashboard analyzes British Airways customer review data. It helps users compare ratings by category, location, aircraft type, and review trends.

Business Question

What areas of the British Airways customer experience are rated highest or lowest, and where should the airline focus improvement efforts?

Key Insights

  • Average rating visuals help compare customer satisfaction across different categories.
  • Location and aircraft filters allow users to explore patterns in review scores.
  • Rating scales should clearly explain what the score is out of, such as out of 5 or out of 10.

Skills & Techniques Used

  • Tableau
  • Data Visualization
  • Interactive Filters
  • Rating Analysis
  • Customer Experience Analysis
  • Dashboard Layout

Design Improvements

To make the dashboard stronger, I would use a more accessible color palette, reduce heavy dark backgrounds, add borders around grouped visuals, and make each rating scale clear to the viewer.

Dashboard Design Approach

Accessibility & Storytelling Focus

Across my dashboard projects, I focus on creating visuals that are not only interactive, but also clear, readable, and useful for business users. I aim to explain what the dashboard is showing, why it matters, and how the insight could support a decision.

Accessibility Practices

  • Use readable labels and clear chart titles.
  • Avoid relying only on color to explain meaning.
  • Use high-contrast colors that are easier to see.
  • Reduce heavy dark backgrounds when a dashboard may be printed.

Storytelling Practices

  • Start each project with the business question.
  • Highlight key insights instead of only showing charts.
  • Use KPI cards, filters, and tooltips to guide the viewer.
  • Connect visuals to real business decisions.
