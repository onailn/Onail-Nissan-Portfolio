Power BI Report: Crime Data In LA

About this Project:

For my 'Crime in LA Dashboard' project, I used Power BI to explore and visualize crime data from Los Angeles between 2020 and 2024. The interactive dashboard highlights trends such as crime over time, crime hotspots, and common weapon types. It includes multiple pages, such as an overview of the data, a deeper crime analysis, and a heatmap showing the most affected areas. The project helped me enhance my skills in data cleaning, using tools like slicers and calculated columns, while also improving the dashboard’s design for better user experience. You can explore the data I used for this project here: Crime Data from 2020 to Present. This project is a great addition to my portfolio and demonstrates my ability to turn raw data into meaningful insights.
