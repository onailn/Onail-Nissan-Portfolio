Power BI Report: Crime Data In LA

About this Project:
For my 'Crime in LA Dashboard' project, I utilized Power BI to explore and visualize crime data from Los Angeles between 2020 and 2024. The interactive dashboard highlights key trends such as crime over time, crime hotspots, and common weapon types. It includes multiple pages, such as:

  • Overview of the data
  • Deeper crime analysis
  • Heatmap showing the most affected areas
This project allowed me to enhance my skills in data cleaning, using advanced Power BI features like:
  • Slicers
  • Calculated columns
  • Dynamic visualizations
I also focused on improving the dashboard’s design for better user experience. You can explore the data I used for this project here: Crime Data from 2020 to Present. This project is a valuable addition to my portfolio and demonstrates my ability to turn raw data into meaningful insights.
